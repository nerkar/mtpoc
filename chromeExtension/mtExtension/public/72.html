Adding the DB2 RDO Entry
Note:
This step is required only if you are licensed for the Abend-AID for CICS DB2
option.
To complete the installation of the Abend-AID for CICS DB2 option, add the required
entry to the CICS system definition dataset (CSD). Member FXRDODB in the installation
sample library (CTL) contains the RDO entry to install the Abend-AID for CICS DB2
option.
Beginning with Release 12.1, the plan name in the bind JCL for Abend-AID for CICS is
CWDB2PLN. If you are running multiple releases of DB2, you must bind a plan for each
release.
Compuware recommends that you add the RDO entry for AADB, specifying
PLAN=CWDB2PLN. Abend-AID for CICS no longer executes DB2 code in the CICS region.
This RDO entry is used as a vehicle to pass the Abend-AID plan name to the TDCAS,
where it is used when creating a thread to DB2 for catalog access.
