Specifying Contact Information
Once you have created a batch dump action definition entry, you can specify detailed
contact information for that entry on the Contact Information screen, as shown in
Figure 21-11 on page 21-12. Users can access the information that you input here by
entering the
N
line command next to the entry for any dump on the Abend-AID
Directory that matches all of the criteria of the action definition.
To access the Contact Information screen, enter the
C
line command next to a batch
dump action definition entry on the Batch Dump Contact Information screen.
