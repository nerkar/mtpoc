Storage Utilization and Memory Requirements
G-5
Use the following formula to estimate the region (in MB) required for the viewing server:
maximum number of concurrent users expected + TP_SUBTASKS number + number of
CICS releases (including those with Japanese language support) installed + 20,
but always specify at least 34MB. Batch Abend-AID only sites can use 0 for the
number of CICS releases.
For example, for 24 users and TP_SUBTASKS set to 7, and 4 CICS releases, use 24 + 7 + 4 +
20 = 55MB.
