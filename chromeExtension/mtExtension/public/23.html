Configuration Checklist
The following steps provide a checklist for configuring Abend-AID. These steps are
discussed in Part 2, “Abend-AID Configuration” of this guide. These steps are required to
get Abend-AID product functionality operating at your site
.
Review Chapter 2, “Preliminary Considerations”.

Ensure “Enterprise Common Components” on page 2-1 is installed.
[ ]
Complete “Step 2. Authorize the Product Target Authorized Libraries” on
page 2-2.
[ ]

page 2-3.

“Step 6. Configure the Abend-AID Viewer” on page 3-1.
[ ]
1. Select the Configure Abend-AID Viewer option from the Abend-AID
Configuration and Customization 16.05 Dialog products configuration
menu.
[ ]
2. Verify/specify the Abend-AID load library dataset name, the Abend-AID for
CICS high-level qualifier name, and the Abend-AID Common Components
nonauthorized load library dataset name.
[ ]
3. Specify the site CICS Transaction Server load and link library dataset
names for all currently supported CICS releases for which your site requires
viewing support in the Abend-AID Viewer.
[ ]
4. Create/specify the site default report shared directory and
report/transaction databases.
[ ]
5. Create/specify the site default source listing shared directory and source
listing databases.
[ ]
6. Specify program compiler listing dataset if using IBM compiler listings for
Abend-AID source support.
[ ]
7. Specify the Abend-AID Viewer customization file name and the Abend-AID
customization load library name, and indicate whether your site uses JES2.
[ ]
8. Generate the Abend-AID Viewer JCL into the Abend-AID installation
sample library (CTL).
