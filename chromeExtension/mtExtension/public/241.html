Specifying the Default Site User Profile
This function is used to set the default Abend-AID Viewer user profile values for all users
at your site. If you don’t modify any of the default profile values, the Compuware-
supplied defaults are used. Individual users can override the values you set here when
they sign onto the Abend-AID Viewer. The user profile function is documented in the
Abend-AID User/Reference Guide
and the
Abend-AID for CICS User’s Guide
.
To set the site user profile, complete the following procedure:
1.
Display the Site User Profile screen, shown in Figure 19-3, using one of two methods:
–
Tab to the Create/Maintain Site User Profile option on the Site Default Options
menu (Figure 19-1 on page 19-1), and then press Enter.
–
Type
USERPROF
or
=1.2
in the COMMAND (or OPTION) field on any
customization
screen and press Enter. You cannot use these fast-path commands
unless you have accessed the customization facility.
