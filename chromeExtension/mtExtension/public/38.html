Migration Considerations
This section discusses the Abend-AID Viewer, batch Abend-AID, and Abend-AID for CICS
migration considerations.
Abend-AID Files
The Abend-AID customization file is now defined with VSAM share options (4 3).
Existing customization files will be altered to have share options (4 3) when being
upgraded. This change allows improved sharing of the Abend-AID customization file.
The following Abend-AID Viewer, batch Abend-AID, and Abend-AID Viewer Release 12.4
files are fully compatible with Abend-AID Release 16.05 and do not have to be redefined
or upgraded:
Viewer-Owned Datasets
•
Dump information file
•
IPCS directory (optional)
•
Persistent data file
•
Viewing server work file
Notes:
1.
Compuware recommends that you back up these Abend-AID Viewer files prior to
migrating to Release 16.05. In the event you need to restore your current release, you
will need to restore these files.
