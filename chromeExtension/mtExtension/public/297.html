Inactive program listing display amount
Applies to COBOL and PL/I Extended Language Support (XLS). Specifies which part
of the program source listing is displayed in the Program Listing section of the
Abend-AID report for the inactive COBOL program(s) or PL/I procedure(s). Valid
entries are:
Abend-AID DD Dynamic Allocation Parameters:
Use a DD as a model
Specifies whether to refer to CEEDUMP, SYSUDUMP, PLIDUMP, or the one that is
specified under USEDDNM, as a model for the statement to be allocated, unless the
model was dynamically allocated. See “Model DD Can Be Dynamically Allocated,”
below.
Model DD can be dynamically allocated
Specifies whether a model is used if the statement was dynamically allocated.
Model DD name to be used
Specifies a DD name to use as a model instead of CEEDUMP, SYSUDUMP, PLIDUMP,
or PL1DUMP. Maximum eight characters. The Default is
blank
.
