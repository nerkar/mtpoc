Assemble and Link-Edit Logon Mode Table
Assemble and link-edit a logon mode table entry to support LU 6.2 communications. See
Figure 16-4 for the required bind image for the logon mode table. This logon mode table
is provided in installation sample library (CTL) member SAMPMOD.
The logon mode table member must be available in SYS1.VTAMLIB or a library in the
SYS1.VTAMLIB concatenation.
