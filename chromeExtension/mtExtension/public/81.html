Database Support Configuration
6-3
Configuring Batch Abend-AID for DB2
Abend-AID for DB2
The batch Abend-AID for DB2 diagnostic modules are included in the Abend-AID Load
Library (SPAALOAD). Follow the instructions in this section to complete the Abend-AID
for DB2 configuration. The remainder of the installation consists of assembling and
linking module CWDB2OPT and binding the Abend-AID plan. Installing stored
procedures support is optional.
CWDB2OPT
Module CWDB2OPT communicates Abend-AID for DB2’s plan name to the Abend-AID
for DB2 diagnostic routines. The plan name can be the recommended default name or a
custom name chosen by you. If your system has multiple DB2 subsystems, Compuware
recommends using the same plan name for all DB2 subsystems. If, however, different
DB2 subsystems require different plan names, each subsystem must have its own
CWDB2OPT module.
CWDB2PLN
Abend-AID for DB2 uses SQL statements to obtain information from the DB2 system
catalog tables when an application’s attach method is CALL, TSO, IMS, or RRSAF attach.
The six DBRMs containing these SQL statements are located in the target sample library
(SKAZSAMP). DBRM members ACDB210, ACDB211, ACDB226, ACDB227, ACDB228, and
ACDB229 are bound into the Abend-AID for DB2 plan (CWDB2PLN) for DB2 supported
versions subsystems.
You can bind these DBRMs into packages prior to binding the plan. This plan name can
be changed, if necessary. If your system has multiple DB2 subsystems, this plan must be
bound on each DB2 subsystem.
In order to obtain the information from the DB2 catalog tables, Abend-AID connects to a
DB2 subsystem. A health check is available to provide information about the Abend-AID
DB2 connections. See Appendix H, “Abend-AID Health Checker” for more detailed
information.
Abend-AID for DB2 Configuration Considerations
Review the following configuration considerations regarding Abend-AID for DB2.
Storage Requirements
When Abend-AID for DB2 is invoked, approximately 132K bytes of storage above the
16-MB line is used.
Accessed DB2 Tables
Abend-AID for DB2 selects from the following DB2 system catalog tables:
SYSIBM.SYSCOLUMNS
SYSIBM.SYSDATATYPES
SYSIBM.SYSDBRM
SYSIBM.SYSFIELDS
SYSIBM.SYSINDEXES
SYSIBM.SYSKEYS
SYSIBM.SYSPACKAGE
SYSIBM.SYSPACKDEP
SYSIBM.SYSPACKLIST
SYSIBM.SYSPACKSTMT
SYSIBM.SYSPLAN
SYSIBM.SYSPLANDEP
SYSIBM.SYSROUTINES
SYSIBM.SYSSTMT
SYSIBM.SYSSYNONYMS
SYSIBM.SYSTRIGGERS
