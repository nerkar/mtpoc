Abend-AID USERMODs
App F
This appendix describes the USERMODs that are available for batch Abend-AID, the
Abend-AID Viewer, and Abend-AID for CICS. These USERMODs are members in the
Abend-AID Common Components target sample library (SKAZSAMP). Review the
comments in each member for complete instructions about applying each USERMOD.
Batch Abend-AID USERMODs
The following USERMODs are intended for use with batch Abend-AID.
USERMOD Job UAASVCD ==> Apply USERMOD UAASVCD
This USERMOD disables the creation of all SVC dumps initiated by batch Abend-AID
dump capture.
USERMOD Job UAA002A ==> Apply USERMOD UAA002A
This USERMOD allows the installation site to change the maximum number of lines
formatted for a single storage section in a COBOL program from the default 200,000
lines.
USERMOD Job UAA003A ==> Apply USERMOD UAA003A
This USERMOD allows installation sites to change which translate table or CCSID is
used when converting the DB2 SQL statement from Unicode to EBCDIC. The default
CCSID is 37. This zap changes it to use 500. This USERMOD was created to address

Abend-AID Viewer USERMODs
The following USERMODs are intended for use with the Abend-AID Viewer.
USERMOD Job UAC001 ===> Apply USERMOD UAC001A
This USERMOD eliminates populating the Comments column on the Abend-AID
Directory screen.
For sites with many thousands of dump entries in the shared
directory, this zap will significantly improve the response time for displaying this
screen.
USERMOD Job UAC002 ==> Apply USERMOD UAC002A
This USERMOD allows the prompt for remote source datasets user ID and password to
be propagated to all remote datasets and associated with the server. User enters user
ID and password only once per remote source listing dataset.
USERMOD Job UAC004 ==> Apply USERMOD UAC004A
This USERMOD is for sites that use a product other than DFHSM to migrate datasets
(for example, CA-DISK) and use VOLSERs other than MIGRAT or ARCIVE to indicate
that a dataset is migrated.
It allows specification of an additional VOLSER that
indicates a dataset is migrated.
This zap applies only to migrated Compuware DDIO
source listing datasets and importing of migrated SYSMDUMP and SVC dump
datasets.
