Batch Abend-AID User-Coded Customization

UKTXT
UKTXT provides a convenient way to display lines of text in an Abend-AID report.
UKTXT can be used only in programs in which macros UKINIT and UKEND are also used.
UKTXT macro syntax
name UKTXT ‘first line of user-coded text’, X
‘next line of user-coded text’
UKEND
UKEND provides program linkage and returns control to either Abend-AID or another
user-coded program. UKEND is used in programs in which macro UKINIT is also used.
UKEND macro syntax
name UKEND MOD=END
UKDUMP
UKDUMP provides a way to display an area of storage in the Abend-AID report. UKDUMP
can be used only in programs in which macros UKINIT and UKEND are also used.
UKDUMP macro syntax
name UKDUMP (address),(length),’optional title’, Z64BIT=NO
UKLIMIT
UKLIMIT limits the amount of storage displayed by a single UKDUMP macro call. If you
use UKLIMIT, place it preceding the UKDUMP macro call for which the limit will be
effective.
name
Optional symbol
user-coded text
Text to be displayed, enclosed in apostrophes. Each line of text is limited to 72
characters.
name
Optional symbol
MOD=END
MOD=program name
Another user-coded program to which control will be passed, or END if control
will be returned to Abend-AID.
name
Optional symbol
address
Register, enclosed in parentheses, that contains the address of the start of the storage
area to display.
length
Register, enclosed in parentheses, that contains the number of bytes of storage to display.
optional title
Text, enclosed in apostrophes, used as a title preceding the displayed storage. This is
optional and can be up to 72 characters in length.
Z64BIT=NO
Z64BIT=YES
When Z64BIT=YES, a 64-bit address is used to indicate the start of the storage area
to be displayed. CWEXIT03 does not need to be run in AMODE=64 to use
Z64BIT=YES. Use of Z64BIT=YES is restricted to z/Architecture systems. The default
is Z64BIT=NO.
