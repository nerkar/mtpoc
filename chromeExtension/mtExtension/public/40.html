Migrating Batch Abend-AID
Follow these steps to update to batch Abend-AID Release 16.05:
1.
Generate the batch Abend-AID 16.05 configuration and customization JCL by
following the procedure in Chapter 4, “Batch Abend-AID Configuration” that uses
your input from the Configure Abend-AID option. Enter the
generate
command on
the last dialog panel.
2.
For JES2 sites only, copy the CWJESTAB and CWJT
xxxx
modules from the previous
CUST.LOAD library to the current CUST.LOAD library.
Note:
If JES has been updated, you must run installation library member $09ASMJT
to create updated CWJESTAB and CWJT
xxxx
modules.
3.
Update existing BDCAS JCL DD statements in installation library member
$68BDCAS:
a.
Update all load libraries to the new Abend-AID 16.05 versions.
b.
Update the customization file name on the FDBDCUST DD statement to the
Abend-AID 16.05 version.
4.
Ensure that all Abend-AID load libraries specified in your site’s linklist have been
updated to Abend-AID version 16.05.
Migrating Abend-AID for CICS
Follow these steps to update to Abend-AID for CICS Release 16.05:
1.
Generate the Abend-AID for CICS 16.05 configuration and customization JCL by
following the procedure in Chapter 5, “Abend-AID for CICS Configuration” that uses
your input from the Configure Abend-AID for CICS option. Enter the
generate
command on the dialog panel.
2.
Update existing TDCAS JCL DD statements in installation library member
$67TDCAS:
a.
Update all load libraries to the new Abend-AID 16.05 versions.
