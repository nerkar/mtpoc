Demonstration Transactions
If you want to use the Abend-AID for CICS COBOL (CCAADEMO), PL/I (CCAADEMP), or
Assembler (CCAADEMA) transaction abend demonstration program, a file entry is
required for the demonstration file. The CCAADEMO, CCAADEMP, CCAADEMA
demonstration file is a VSAM KSDS. Installation sample library (CTL) member FXRDOxxx
contains the RDO entry. “Setting Up the Transaction Abend Demonstration Programs” on
page 5-10 provides more information about installing and using the demonstration
transactions.
Using Resource Definition Online or the CSD Utility DFHCSDUP
You can use a CEDA transaction to add the Abend-AID for CICS entries to a CICS system
definition (CSD) file. Alternatively, you can also use the CSD file utility DFHCSDUP to
add the definitions in batch.
The resource definition online (RDO) statements for the CICS table entries for Abend-AID
for CICS are contained in members in the Abend-AID installation sample library (CTL).
Table 5-1 provides the member names containing the RDO definitions. The RDO
commands and JCL are provided in the members.
Installing the Abend-AID for CICS Group
If you have created a new group for
the Abend-AID for CICS programs and transactions,
you can add the programs and transactions to an active CICS system by entering the
following CEDA command:
INSTALL GROUP(FXGROUP)
Alternatively, add group FXGROUP to one of the lists specified in the CICS startup
GRPLIST.
