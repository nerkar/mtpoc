Two Viewing Servers and Two TDCASs
This section provides a configuration example of two viewing servers and two TDCASs on
a single MVS image. It describes the datasets used by the viewing servers and TDCASs. It
does not provide a viewing access configuration example since the number of TDCASs
does not affect viewing access configuration. See Figure D-2 on page D-5 for an
illustration of viewing access configuration.
The information in this section is useful if you have only one MVS image but want to
configure more than one Abend-AID viewing server, and you have a requirement for
more than one TDCAS.
After you create your first viewing server, you can easily configure additional ones by
repeating the steps described in Chapter 3, “Abend-AID Viewer Configuration”. Create a
second TDCAS by repeating the steps described in Chapter 12, “Transaction Dump
Capture Address Space Customization”.
