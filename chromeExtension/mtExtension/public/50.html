8. Allocate the Viewing Server Work File ($63WORK)
The viewing server work file is a VSAM RRDS. You must allocate one viewing server work
file per Abend-AID server, and you cannot share viewing server work files between
viewing servers.
Member $63WORK in the installation sample library (CTL) contains a job to allocate the
viewing server work file using IDCAMS. Before you run $63WORK, make the following
modifications:
1.
Change each occurrence (2) of the viewing server work file dataset name to the name
you choose for the file.
2.
If applicable, verify that the volume serial number or SMS storage and management
class conform to your site standard.
9. Allocate and Populate the TSO/ISPF Access Facility Dataset ($65SCPY)
Run the job, installation sample library (CTL) member $65SCPY, to create and populate
the TSO/ISPF access facility dataset. This file must be allocated before the Abend-AID
Viewer can be successfully started.
Job $65SCPY is generated automatically. Before you run the job, ensure that the correct
library names are specified and that the name of the TSO/ISPF access facility dataset to be
created conforms to your site's dataset naming conventions.
10. Allocate the Scratchpad Listing File ($65SCRT)
Run the job, installation sample library (CTL) member $65SCRT, to create the Scratchpad
listing file. The Scratchpad listing file is required.
Job $65SCRT is generated automatically. Before you run the job, ensure that the correct
library names are specified and that the name of the Scratchpad dataset to be created
conforms to your site’s dataset naming conventions. When the dataset is created, it will
be set up using Compuware’s recommended parameters since this DDIO file is needed to
support the conversion of program compiler listings to Compuware source listings within
the viewer.
Linking the CICS Trace Formatting Module (CxxxLTRC)
Note:
If you’re installing batch Abend-AID only, skip to “Verifying Viewing Server
External Security Authority”.
Installation sample library members CxxxLTRC contain sample JCL to link a copy of the
CICS trace formatting module for the CICS releases into the corresponding Abend-AID
for CICS module. The Abend-AID for CICS versions of the trace formatting modules
reside in the Abend-AID for CICS nonauthorized load library (SKFXLOAD).
The xxx in the member names represents the release level of CICS.
