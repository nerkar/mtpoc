Viewing Access Customization
Chap 16
Note:
You will be able to configure CICS access only if your site is also licensed for and
has installed Abend-AID for CICS.
You can configure a viewing server to support viewing access from VTAM, TSO/ISPF, CICS
(local or remote), the Abend-AID Web Browser Interface, or any combination of the four
methods concurrently. Each method has different configuration requirements external to
Abend-AID (for example, VTAM APPLIDs or ISPF panel modifications).
This chapter describes the steps required to configure viewing access from VTAM,
TSO/ISPF, CICS (local or remote), and the Abend-AID Web Browser Interface. Remember
that a viewing server can support more than one viewing option; simply perform the
steps for each method you require for the viewing server.
Because Distributed Viewing Support (DVS) is optional and requires that you configure at
least one of the other viewing access methods first, Compuware recommends that you
add DVS after you’ve completed the Abend-AID/Abend-AID for CICS basic installation
procedures. After completing these procedures, see “Configuring Optional DVS with the
Abend-AID Viewer” on page 16-7 for the configuration procedure for adding DVS.
Some of the requirements described in this chapter apply to more than one viewing
access method. Table 16-1 indicates the viewing access requirements, and to what
method they apply.
If a requirement supports more than one viewing access method,
you need to perform it only once per viewing server.
