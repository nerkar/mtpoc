The Region Configuration screen displays the following input fields:
Job Name
Specifies the name of the MVS address space. You can use the asterisk (*) wildcard
character to create a generic job name for multiple regions.
If you specify anything other than asterisk (*) in this field, then you
must
install
the MVS post-dump exit.
Chapter 17, “Automatic SVC Dump Processing Customization” contains
information about the choices that are available to you for processing region
dumps.
CICS APPLID
Specifies the CICS APPLID for this region. You can use the asterisk (*) wildcard
character to create a generic APPLID.
In a non-XRF environment, CICS APPLID is the generic APPLID for both region
and transaction dumps. In an XRF environment, for transaction dumps, the
APPLID is the generic CICS APPLID. For region dumps, it is the specific CICS
APPLID.
MVS SYSID
Specifies the MVS SYSID of the executing CICS region. You can use the asterisk
(*) wildcard character to create a generic SYSID.
Rgn Dump Profile
Specifies the region dump capture profile for this entry. This profile indicates
automatic processing choices for region dumps. See Chapter 17, “Automatic SVC
Dump Processing Customization” for more information. “Modifying Region
Dump Profiles” on page 20-14 describes creating your own profile members.
$DEFAULT indicates the Compuware-supplied defaults are used.
Tran Dump Profile
For Abend-AID for CICS only. Specifies the transaction dump capture profile for
this entry. This profile indicates automatic processing choices for transaction
dumps. See “Modifying Permanent CICS Transaction Dump Profiles” on page
20-18 for more information. This section describes creating your own profile
members. $DEFAULT indicates the Compuware-supplied defaults are used.
Tran Dump Globals
For Abend-AID for CICS only. Specifies the transaction dump global options
member for this entry. This profile indicates global processing choices for
transaction dumps. See “Modifying CICS Transaction Dump Global Options” on
page 20-25 for more information. $DEFAULT indicates the Compuware-supplied
defaults are used.
Tran Dump Handlers
For Abend-AID for CICS only. Specifies the transaction dump handlers profile for
this entry. This profile lists programs that routinely handle abends/conditions
and that will not be identified as the abending program. Instead, the program
calling these handlers is identified. $DEFAULT indicates the Compuware-
supplied defaults are used. You can also create your own site-defined abend
handler profile(s). See “Modifying CICS Transaction Dump Handler Profiles” on
page 20-35 for more information. If this field is left blank, Abend-AID for CICS
makes no attempt to identify and ignore any abend- handling programs.
