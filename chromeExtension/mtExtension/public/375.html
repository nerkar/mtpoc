There is a TDCAS running on each MVS image. You cannot share a TDCAS across
MVS images. If you want to run more than one TDCAS per image, see “Two Viewing
Servers and Two TDCASs” on page D-6.
The TDCAS writes to only the shared directory and transaction databases. It does not
read from or write to the other viewing server-owned datasets.
