The statement used for Abend-AID Viewer access is:
AA,’CMD(%$66CLST)’
Note:
Change AA to the letter or number you have chosen, if applicable. Also, if you
renamed $66CLST, ensure that you reflect the correct name in the CMD string.
You must also code the following statement on the ISPF panel you choose, and place it
after the &ZSEL variable:
&ZTRAIL = .TRAIL
Notes:
1.
Begin the &ZTRAIL statement in column 1.
2.
If you receive message ERWIN0006 trying to access Abend-AID, make sure that the
variables reserved by ISPF (that is, ZCMD, ZSEL, and ZTRAIL) are used correctly.
3.
You must exit and reaccess ISPF for the newly added options to be usable.
4.
The dataset names must be in uppercase.
