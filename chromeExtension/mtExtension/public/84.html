2. Verify and Submit the JCL to be Used for Abend-AID’s DB2 Support
JCL member $21
nxxxx
, generated in the Abend-AID installation library, configures
Abend-AID’s DB2 support. Based on the information input during the generation of
the JCL, the member created will install, assemble and link module CWDB2OPT, and
bind the Abend-AID plan using PLAN or PLAN PACKAGE support. Installation sample
library (CTL) JCL member CWDB2OAS has been included in the event that
CWDB2OPT needs to be assembled and linked outside of the $21nxxxx JCL job.
Abend-AID for DB2 Package Versioning
Abend-AID for DB2 now uses package versioning in its precompile preparation
process. To take advantage of this method, the Abend-AID for DB2 DBRMs must be
bound into packages at installation time. When maintenance is applied, use
installation sample library (CTL) member $22
nxxxx
, where
n
represents the one-
numeric character DB2 release identifier and
xxxx
is the four-alphanumeric character
DB2 subsystem ID. The replacement DBRMs are bound using the same collection ID
and package names as those used during the installation, allowing DB2 to locate the
needed package during processing. This method also eliminates the need to rebind
the plan after the latest packages are bound.
