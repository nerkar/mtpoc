3. To enable Abend-AID support for DB2, select the DB2 Support option from
the Database Support menu.
a.
Add/edit an entry for each DB2 release that requires Abend-
AID/Abend-AID for CICS support at your site by completing the
screen fields.
b.
Generate the installation JCL.
c.
Review batch Abend-AID/Abend-AID for CICS for DB2 security
considerations.
d.
Perform DB2 binds ($21
nxxxx
).
e.
Add an RDO Entry (Abend-AID for CICS only).
f.
Specify the DB2 handle routine (optional for Abend-AID for CICS
only).
g.
Test batch Abend-AID/Abend-AID for CICS DB2 diagnostics.
[ ]
4. To enable Abend-AID support for IDMS, select the IDMS Support option
from the Database Support menu.
a.
Enter site IDMS and job card information.
b.
Generate the IDMS installation JCL.
c.
Run $31IDMS1.
d.
Relink COBOL programs that access a CA-IDMS 16.0 or more current
database that are still linked with CA-IDMS 10.2 or less current
subschemas.
e.
Relink COBOL programs for which Abend-AID is unable to provide
CA-IDMS database diagnostic information.
f.
Relink all non-COBOL programs.
