creates JCL for the all CICS releases that this Abend-AID release supports. From the
Abend-AID Configuration and Customization Dialog 16.5 product menu (Figure 2-5 on
page 2-6), select the Configure Abend-AID for CICS option. The screen shown in Figure 5-
1 is displayed.
Step 8. Configure Abend-AID for CICS
From the Abend-AID Configuration and Customization Dialog 16.5 products
configuration menu (Figure 2-5 on page 2-6), select the Configure Abend-AID for CICS
option to display the screen shown in Figure 5-1.
Complete the library name fields that are appropriate for each currently supported
release of CICS Transaction Server for which your site requires viewing support in the
Abend-AID Viewer.
