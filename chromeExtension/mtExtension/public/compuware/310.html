Defining a Default Remote Server
The Remote Server Definition screen, shown in Figure 20-33, lets you enter one remote
server name to be used a default when the IDR source file name is not found on the local
system. Display this screen using one of two methods:
•
Tab to the Remote Server Definition option on the Dump Capture and Processing
Menu (Figure 20-1 on page 20-3) and then press Enter.
•
Type
REMSRVR
or
7
in the COMMAND field on the Dump Capture and Processing
Menu and then press Enter.
Figure
20-33.
Remote Server Information Screen
This screen displays the current viewing server name. You can specify a remote server
name to be used as the default for IDR source files names that are not resident on the
current viewing server.
Defining Remote Source Listings to Remote Servers
The Remote Source Information screen, shown in Figure 20-34, lets you define remote
source datasets to remote servers. Display this screen using one of two methods:
•
Tab to the Remote Source Information option on the Dump Capture and Processing
Menu (Figure 20-1 on page 20-3) and then press Enter.
•
Type

or

in the COMMAND field on the Dump Capture and Processing
