Executing the IVPs and Demonstration Transactions
Execute one or more of the following members in the Abend-AID installation sample
library (CTL). These jobs compile listings to the source listing database using the
following language postprocessors:
•
$79KXCOB for the COBOL postprocessor
•
$79KXPLI for the PL/I postprocessor
•
$79KXASM for the Assembler postprocessor
•
JCLMQ4 for Abend-AID for WebSphere MQ support with the COBOL postprocessor.
Note:
Ensure that the load libraries you specify on the SYSLMOD DD in the sample JCL
are included in your CICS DFHRPL library concatenation.
Setting Up the Transaction Abend Demonstration Programs
The Abend-AID for CICS COBOL (CCAADEMO, CFDB2IVP, CFDB2IV2, and CCAADEME),
PL/I (CCAADEMP), and Assembler (CCAADEMA) demonstration programs located in the
installation sample library (CTL) illustrate the transaction abend support provided by
Abend-AID for CICS. They use a small VSAM file that is created by the process described
in this step. You can delete the file when you no longer need it for demonstration
purposes.
To activate the CCAADEMO, CFDB2IVP, CFDB2IV2, CCAADEMP, CCAADEMA, or
CCAADEME program, complete the following procedure:
1.
Allocate and load the demonstration file.
Creating this file allows you to use the CCAADEMO, CFDB2IVP, CFDB2IV2,
CCAADEMP, or CCAADEMA demonstration program. Modify and run installation
sample library (CTL) member CCADEFEM to allocate the VSAM KSDS file and load it
with five records.
Note:
A return code of 4 from CCADEFEM is acceptable. If you receive a return code
greater than 4, resolve the reason for the return code.
2.
Compile and link-edit the test program.
