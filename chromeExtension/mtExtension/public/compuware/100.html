Abend-AID Installation and Customization Guide
Chapter 17, “Automatic SVC Dump Processing Customization”:
Chapter 17 describes how to customize the Abend-AID Viewer to
automatically import SVC dumps.
Chapter 18, “Accessing the Site Customization Options”:
Chapter 18 describes how to access the Abend-AID Viewer site customization
options.
Chapter 19, “Site Defaults Customization”:
Chapter 19 describes how to set the site defaults for Abend-AID Viewer
screen print output, the user profile, screen attributes, and PF keys.
Chapter 20, “Dump Capture and Processing Options Customization”:
Chapter 20 describes the steps for modifying the Abend-AID Viewer dump
capture support to your site’s requirements.
Chapter 21, “Action Definitions”:
Chapter 21 describes how to provide contact information when batch,
transaction, and SVC dumps occur.
Chapter 22, “User-Defined DSECTs”:
Chapter 22 describes how to create user-defined DSECT files.
Chapter 23, “External Security Considerations”:
Chapter 23 describes Abend-AID external security considerations and how to
implement the external security interface for user functions.
