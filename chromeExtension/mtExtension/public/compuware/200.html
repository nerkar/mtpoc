The DSECT file used by an Abend-AID for CICS viewer contains the CICS and
MVS DSECTS if the $61DSECT and $61MVSD jobs were run during the Abend-

f.
Verify that the //FDBDCUST DD in the viewing server startup JCL points to a
single customization file. Choose the existing customization file that has the
most extensive updates. Manually enter the updates from the other
customization file into the Abend-AID Viewer customization file.
Customizing Viewing Access
You can configure a viewing server to support viewing access from VTAM, TSO/ISPF, CICS
(local or remote), the Abend-AID Web Browser Interface, or any combination of the four
methods concurrently. Each method has different configuration requirements external to
Abend-AID (for example, VTAM APPLIDs or ISPF panel modifications). See the following
sections in Chapter 16, “Viewing Access Customization”for information about the
viewing access method you want to customize:
•
“Configuring VTAM Access” on page 16-2
•
“Configuring TSO/ISPF Access” on page 16-3
•
“Configuring CICS Access” on page 16-16
•
“Configuring Abend-AID Web Browser Interface Access” on page 16-20.
