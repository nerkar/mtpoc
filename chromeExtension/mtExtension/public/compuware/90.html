3. Add an RDO Entry
The Abend-AID for CICS DB2 option requires an entry in the CICS system definition
dataset (CSD). “Adding the DB2 RDO Entry” on page 5-8 describes adding the entry.
If you haven’t already done so, add the entry now.
4. Specify the DB2 Handle Routine (Optional)
The Abend-AID for CICS DB2 option generates output for non-zero SQL return codes.
However, you must specify the action taken when an SQL code is returned. For non-
zero SQL return codes, an EXEC CICS DUMP or EXEC CICS ABEND must be issued to
generate the Abend-AID for CICS information. The following example produces an
Abend-AID for CICS analysis and continues processing. You may want to distribute a
copy of this example to your site’s application programmers.
EXEC SQL WHENEVER SQLERROR
GOTO 9998-HANDLE-DB2-ERROR
END-EXEC.
