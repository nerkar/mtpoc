Region Dump Demonstration Transaction
The region dump demonstration transaction is ERWV. To use the demonstration
transaction for CICS region dump support, you must have included the appropriate CICS
table entries for the demonstration transaction. The CICS table entries are described in
“Modifying CICS Resource Entries” on page 5-3.
ERWV causes a storage violation that results in a CICS region dump. Chapter 16,
“Analyzing Storage Violations,” of the
Abend-AID for CICS User’s Guide
describes using
Abend-AID for CICS to resolve the storage violation.
To use the region dump demonstration transaction, enter the ERWV transaction from
CICS.
