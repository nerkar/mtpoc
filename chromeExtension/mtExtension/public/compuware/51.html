Abend-AID Viewer Configuration
3-9
Review, modify, and submit this JCL for each CICS Transaction Server release for which
your site requires viewing support in the Abend-AID Viewer. Failure to do so causes the
Abend-AID for CICS trace displays to be unavailable. Ensure that:
•
The //TLIB DD statement points to your CICS load library for this release of CICS.
•
The //SYSLMOD DD statement points to the Abend-AID for CICS nonauthorized load
library (SKFXLOAD).
Notes:
1.
Message IEW2646W is normal, and can be ignored.
2.
If you apply IBM maintenance to the CICS trace formatting module, you must
run CxxxLTRC again.
3.
Place modules DFHTTxxx, where xxx is the CICS release, in a system link list library.
Failure to do so results in messages CSV003I, CSV031I, and IEW4009I when viewing
Abend-AID for CICS trace displays.
Verifying Viewing Server External Security Authority
If your site uses an external security package, updates may be required in order to allow
the Abend-AID Viewer to execute successfully. See “External Security Requirements for
All Sites” on page 23-1 and “Abend-AID Viewer Security Requirements” on page 23-2 for
more information. If your site uses ACF2 or Top Secret as your security package, also
review “Enabling RACROUTE under ACF2 and Top Secret” on page 23-15 for required
SAFDEF information.
Starting the Viewing Server
You need to start the viewing server in order to use the Abend-AID Viewer or Abend-AID
for CICS. Compuware recommends doing so at this point in order to validate your
configuration.
Note:
You can start the viewing server now, even though you have not yet configured
Abend-AID report routing or the global options.
To start the viewing server, submit the Installation sample library (CTL) member
$65SRVR job. Appendix A, “Controlling the Abend-AID Viewing Server, BDCAS, and
TDCAS” contains more information about starting the viewing server. The
Abend-AID
User/Reference Guide
and the
Abend-AID for CICS User’s Guide
describe logging onto the
viewing server.
Sharing a Viewing Server with Abend-AID for CICS
If your site is licensed for both Abend-AID and Abend-AID for CICS, you can use the
Abend-AID Viewer to view both batch and CICS transaction reports in a single viewing
server. See “Sharing a Viewing Server with Abend-AID for CICS” on page 15-21 for the
steps to complete this optional configuration.
