Customization Options Menu
Note:
You cannot execute non-customization fast-path commands from the
customization screens. For example, you cannot execute the DIAG fast-path
command from the customization screens. Further, you cannot execute
customization fast-path commands from the Abend-AID dump analysis screens.
From this menu, you can access the customization screens from the following menus:
•
Site Default Options Menu
Use this menu to access the customization screens for setting your site defaults for
screen print output, user profile, and screen attributes, as described in Chapter 19,
“Site Defaults Customization”.
•
Dump Capture and Processing Menu
Use this menu to access the customization screens for setting your site defaults for
Summary screen configuration, region dump capture profiles, and dump notification
information, as described in Chapter 20, “Dump Capture and Processing Options
Customization”.
•
Notify Users and Execute Scripts
Use this menu to create contact information definitions for batch dumps, as
described in Chapter 21, “Action Definitions”.
•
Create/Maintain Site News
Use this screen to enter site-defined news, as described in “Creating/Maintaining Site
News” below.
•
Create/Maintain User-Defined Diagnostics
Use this screen to enter user-defined diagnostics for common or frequently occurring
site abends/faults, as described in “Creating/Maintaining User-Defined Diagnostics”
on page 18-3.
•
Maintain Workbench Host/Port Table
Use this screen to maintain the Topaz Workbench host/port table, as described in
“Maintaining the Topaz Workbench Host/Port Table” on page 18-4.
Creating/Maintaining Site News
Use the Site News Menu, shown in Figure 18-3 on page 18-3, to access screens that enable
you to add site-specific information to the panels that display the Abend-AID and Abend-
AID for CICS product news in the Abend-AID Viewer via the NEWS fast-path command.
