Preventive Maintenance
Chap 24
For Abend-AID Release 16.5, refer to the Maintenance chapter of the
Compuware Installer
Mainframe Products SMP/E installation Guide
for the procedures to install preventive
maintenance using SMP/E. This document was attached to the e-mail you were sent
regarding Receive From Network (RFN) using SMP/E for your order of Compuware
mainframe products. This document is also included on the EP media browser for
Enterprise Common Components (ECC) Release 3.3 and on the ECC product
documentation site on FrontLine.
CAUTION:
Products using address spaces or server regions where Abend-AID and Abend-AID for
CICS are allocated must be shut down while maintenance is being applied to their
libraries and restarted after the maintenance process is complete. After completing
the SMP/E APPLY step, but before the ACCEPT step, as described in the Maintenance
chapter of the
Compuware Installer Mainframe Products SMP/E installation Guide
,
return to this page and do the following:
Stop and Restart Abend-AID
After you complete the SMP/E APPLY step using member $15MAPLY, do the following to
enable the new maintenance for batch Abend-AID only:
1.
Stop the SVC51 interface.
2.
Shut down all batch dump capture address spaces (BDCASs).
3.
Restart all batch dump capture address spaces (BDCASs).
4.
Restart the SVC51 interface.
If you’re applying maintenance to batch Abend-AID with the Abend-AID Viewer
installed, in addition to the above steps, do the following Steps 1 and 3 through 5:
Or, if you’re applying maintenance to Abend-AID for CICS, do the following to enable
the new maintenance:
1.
Shut down all Abend-AID viewing servers if you have not already done so.
2.
Shut down all Abend-AID for CICS transaction dump capture address spaces
(TDCASs) if you have not already done so.
3.
If the maintenance affects the MVS post-dump exit, make sure you complete the
HOLDDATA instructions regarding the post-dump exit.
4.
If you are licensed for and have installed the Abend-AID or Abend-AID for CICS DB2
option and the maintenance affects the DB2 option, make sure you complete the
HOLDDATA instructions regarding the DB2 DBRM or plan/package.
5.
Restart the Abend-AID viewing servers.
6.
Restart the Abend-AID for CICS TDCASs.
7.
Continue with the SMP/E ACCEPT step using member $16MACPT as described in
Compuware Installer Mainframe Products SMP/E installation Guide.
