For ease of configuration, Compuware recommends that you define each viewing server on an MVS

name
Specifies a four-character MVS subsystem name. Abend-AID dynamically starts the
subsystem, so you must not add an entry for it to SYS1.PARMLIB(IEFSSNxx). The
name you specify depends on how this viewing server is using the subsystem, as
follows:
–
If this is the only viewing server on the MVS image, the name you specify must
be unique on this MVS image.
–
If you have multiple viewing servers on the MVS image but are not sharing the
subsystem between them, the name you specify must be unique on this MVS
image.
–
If you are sharing the subsystem between multiple viewing servers on this MVS
image, the name you specify must be the same for all the viewing servers on the
image. The name you specify must not be used by any other non-Abend-AID
viewing subsystem on this MVS image.

Parameters Required to Support CICS Remote Viewing Access
The parameters listed below must be specified if you are configuring this viewing server
for CICS remote viewing access.
LU62_APPLID=applid
Specifies the one- to eight-character VTAM LU 6.2 APPLID name defined for use by
this viewing server to support CICS remote viewing access. See “Configuring VTAM
Access” on page 16-2 and “Installing CICS Remote Viewing Access” on page 16-17 for
a description of the requirements for the LU 6.2 APPLID. There is no default value for
this parameter.
