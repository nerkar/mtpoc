Preliminary Considerations
Chap 2
This chapter describes the preliminary considerations before you begin the Abend-AID
configuration procedures.
Enterprise Common Components
Enterprise Common Components (ECC) is an umbrella program that contains several
components including Compuware Shared Services (CSS), License Management System
(LMS), Host Communications Interface (HCI), Compuware Mainframe Services
Controller (CMSC), and Base Services.
Abend-AID operation requires the ECC components, which are provided on separate
installation media from the Abend-AID product. ECC has its own installation procedure,
which you perform in conjunction with the Abend-AID installation. Review the ECC user
documentation before installing or reinstalling Abend-AID.
Abend-AID Release 16.05 requires the following ECC 16.05 software, with all current
maintenance applied:
•
Compuware Shared Services (CSS) Release 16.05 or more current
•
License Management System Release (LMS) 16.05 or more current
•
Compuware Host Communications Interface (HCI) Release 16.05 or more current
•
Base Services 16.05 or more current
•
Compuware Mainfame Services Controller (CMSC) 16.05 or more current.
Notes:
1.
With Release 16.05, in order to simplify Enterprise Common Components (ECC)
SMP/E installation, Base Services, CSS, LMS, HCI, and CMSC modules are now
installed as a single ECC FMID. These components have been consolidated into the
SLCXxxxx libraries. Ensure that you change all references to the old Compuware HCI
libraries (SLHCAUTH and SLHCLOAD) to the ECC 16.05 libraries (SLCXAUTH and
SLCXLOAD) in your installation JCL, especially viewing server JCL when it is
converted to 16.05.
2.
Enterprise Common Components (ECC) Release 16.5 or more current cannot be in
the same zones as Abend-AID 12.4 and prior releases.
3.
If you plan to use COBOL 5.1/5.2 support, you must install and use CSS Release 16.05
or more current.
4.
If you want to use Internet Protocol Version 6 (IPV6) support, you must install and
configure HCI 16.05 or more current.
5.
For installing the product from the installation media, Abend-AID requires Internet
Explorer 6.0 or more current, or other equivalent browser. For viewing product
documentation PDFs, Adobe Reader 6.0 or more current is required.
