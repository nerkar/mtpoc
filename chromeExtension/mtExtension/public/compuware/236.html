Authorized users can use the following line commands to display the User-Defined
Diagnostics input screen, as shown in Figure 18-6:
•
I (Insert):
Use this line command to display the User Defined Diagnostics screen
with all input fields blank.
•
R (Replicate):
Use this line command to display the selected entry on the User
Defined Diagnostics screen with all input fields blank except the User defined
Diagnostic Text field.
•
D (Delete):
Use this line command to delete the selected entry.
•
U (Update):
Use this line command to display the selected entry on the User Defined
Diagnostics screen with all input fields prefilled. Overtype the fields you want to
change.
•
S (Display):
Use this line command to display the selected entry on the User Defined
Diagnostics screen with all input fields prefilled. This line command is for display
purposes only and does not let you save any changes.

Maintaining the Topaz Workbench Host/Port Table
The Workbench Host/Port Table screen, shown in Figure 18-7 on page 18-5, enables
authorized users to maintain host name/port number information for establishing a
connection to the Topaz Workbench (formerly, Compuware Workbench). Users are
granted security authorization to this screen with access to the Customizations Options
menu (Figure 18-2 on page 18-2) through the CUSTOM subparameter of the
EXTERNAL_SECURITY_FUNCTION_CHECK viewing server configuration parameter, as
described in “Parameters to Enable External Security for Viewing Servers” on page 15-8.
