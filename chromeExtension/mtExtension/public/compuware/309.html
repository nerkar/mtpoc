Dump Capture and Processing Options Customization
20-61
Ignore Abend
Specifies whether Abend-AID is to ignore abends that meet the specified criteria.
Valid values are:
N:
Default. Abend-AID processes this abend.
Y:
Abend-AID ignores this abend.
Suppr Dups
Specifies whether to suppress duplicate dumps that meet the criteria. History
information for the 100 most recent duplicates is kept. Valid values are:
N:
Default. Do not suppress duplicate dumps.
Y:
Suppress all duplicate dumps.
Note:
The Suppr Dups option is available only when the Ignore Abend option is set
to
N
.

then

duplicate of one that has already been processed by Abend-AID.
The other two condition statements indicate what action is taken

by user ID TSOUSER3 abends, and
IF
a program with JOBNAME TEST abends.
