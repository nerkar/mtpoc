Common DB2 Installation Procedure
Beginning with Release 12.1, the installation dialog shares the same panels for installing
the Abend-AID or Abend-AID for CICS optional DB2 licensed support. The default
Compuware-supplied plan name for both products has been changed to
CWDB2PLN
. You
can change this default plan name in the installation dialog. However, Compuware
recommends keeping the same plan name for both products so that you can exploit the
common maintenance enhancement described in the next paragraph. The DBRMs used
to bind packages/plans for both product have been changed to
ACDB210
,
ACDB211
,
ACDB226
,
ACDB227
,
ACDB228
, and
ACDB229
, which are located in the target sample
library (SKAZSAMP). Both products use installation sample library (CTL) JCL members
$21
nxxxx
to perform the binds.
Like with Abend-AID DB2 processing which occurs in the Batch Dump Capture Address
Space (BDCAS), Abend-AID for CICS DB2 processing has been moved to the Transaction
Dump Capture Address Space (TDCAS). This change has a positive impact on
maintenance because only one set of modules accesses DB2. Problem fixes will apply to
both Abend-AID and Abend-AID for CICS products with one PTF.
