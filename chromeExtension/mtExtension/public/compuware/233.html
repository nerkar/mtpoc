Accessing the Site Customization Options
Chap 18
This chapter explains how to access the Abend-AID Viewer site customization options.
Accessing the User Controls Menu
The Abend-AID Viewer site customization options are available as a selection on the User
Controls menu. Display the User Controls menu, shown in Figure 18-1, using one of the
following methods:
•
If you do not currently have a dump selected, type
USER
in the COMMAND field of
the Fault Summary or Abend-AID Directory display, and then press Enter.
•
If you currently have a dump selected, use one of two methods:
–
Tab to User Control Facility on any Abend-AID Viewer Primary Options menu,
and then press Enter, or
–
Type
USER
or
=U
in the COMMAND (or OPTION) field on any screen and press
Enter.

Accessing the Customization Options Menu
Note:
You can restrict access to the customization screens by specifying the CUSTOM
subparameter of the EXTERNAL_SECURITY_FUNCTION_CHECK viewing server
configuration parameter, as described in “Parameters to Enable External Security
for Viewing Servers” on page 15-8. You must also write the appropriate rules for
your external security package, as described in “Controlling User Access to
Specific Functions” on page 23-8.
Display the Customization Options menu, shown in Figure 18-2 on page 18-2, using one
of two methods:
•
Tab to Site Customization Options on the User Controls menu, and then press Enter.
•
Type
CUSTOM
or
USER.5
in the COMMAND (or OPTION) field on any screen and
press Enter. You can also type
=U.5
if you currently have a dump selected.
