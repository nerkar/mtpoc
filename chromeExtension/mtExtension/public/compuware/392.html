Abend-AID Installation and Customization Guide
Virtual Memory Requirements for the TDCAS
The transaction dump capture address space (TDCAS) controls transaction dump capture
and processing. The virtual storage requirements for the TDCAS are shown in Table G-4.
Virtual Memory Requirements for the Viewing Server
The viewing server is a VTAM application that runs in its own address space. It controls
Abend-AID Viewer access to the files and user access to analysis and dump information.
The viewing server virtual memory requirements are shown in Table G-5.
