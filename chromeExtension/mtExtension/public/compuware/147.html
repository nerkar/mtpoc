Customizing Batch Abend-AID Additional Facilities
If you want to implement batch Abend-AID HOTKEY command support for your site,
continue with “Setting Up the HOTKEY Command” below.
Setting Up the HOTKEY Command
This procedure creates a site command that your Abend-AID users can specify to access
Abend-AID reports from anywhere in ISPF. The command is added to the ISPF Site
Command Table.
Notes:
1.
Because the one-key access program, #XAAHTKY, is accessed via ISPF dialog
management, in order to ensure TSO user access to Abend-AID reports from
anywhere within ISPF, Compuware recommends that you place #XAAHTKY in a Link
Pack Area (LPA) library.
2.
Use of this function is restricted by default in the viewing server parameters. To
remove this restriction, change HOT_KEY_ACCESS_RESTRICTED to
NO
, as described
in “Parameter to Restrict the HOTKEY Command Users” on page 15-15.
1. Define the HOTKEY command and action
This site-defined command invokes the specified Abend-AID HOTKEY action. The default
command is
AAHOTKEY.
However, your TSO/ISPF systems programmer can use
AAHOTKEY
as the HOTKEY command or some other character string that conforms to
ISPF site command naming requirements.
Note:
If your site also has Xpediter/TSO installed, do
not
define the HOTKEY command
as only the two characters
AA
. Doing so may interfere with the execution of the
Xpediter/TSO AA SNAP command.
The Abend-AID HOTKEY action can be set up at your site using
one
of the two following
formats:
Format 1: SELECT CMD(%AAVIEW)
Format 2: SELECT CMD(EX 'VIEWER.INSTALLATION.FDBDCLST(AAVIEW)')
