Abend-AID Installation and Customization Guide
Abend-AID for CICS USERMODs
The following USERMODs are intended for use with Abend-AID for CICS.
USERMOD Job UFX001 ==> Apply USERMOD UFX001A
This USERMOD serializes the CREATE-MEM processing in CTCJDCP1.
USERMOD Job UFX002 ==> Apply USERMOD UFX002X
This USERMOD allows CTCTC4800I debugging messages to be displayed.
These may
help debug DB2 installation issues.
USERMOD Job UFX003 ==> Apply USERMOD UFX003A
This USERMOD allows the installation site to change the name of the procedure
submitted by the MVS Post-Dump Exit.
This fix may be different for each customer
depending on the length of the procedure name.
Padding with blanks must be done
on the left side of the name for the START command to work. For example,
'MSDCCOPY' must become ' XXXXXXX' for a seven-character proc name.
USERMOD Job UFX011 ==> Apply USERMOD UFX011Z
This USERMOD issues RACROUTE checks under the Abend-AID Viewer ACEE instead
of the user's ACEE even if external security is enabled.
USERMOD Job UFX013 ==> Apply USERMOD UFX013X
This USERMOD results in all messages generated in Abend-AID for CICS being
uppercased to assure readability in a DBCS Japanese language environment.
USERMOD Job UFX017 ==> Apply USERMOD UFX017X
This USERMOD reverses standard processing when processing an Abend-AID for CICS
transaction dump and reports on the most current TACB, instead of the first or oldest
TACB.
USERMOD Job UFX018 ==> Apply USERMOD UFX018A
This USERMOD enables the installation site to change the maximum number of lines
formatted for each storage section in a COBOL program from the default 200,000
lines.
USERMOD Job UFX019 ==> Apply USERMOD UFX019X
This USERMOD allows installation sites to change which translate table or CCSID is
used when converting the DB2 SQL statement from Unicode to EBCDIC. The default
CCSID is 37. This zap changes it to use 500. This USERMOD was created to address

USERMOD Job UMP001 ==> Apply USERMOD UMP001A
This USERMOD activates the option to have an enqueue report created for either an
S122 or S222 abend.
