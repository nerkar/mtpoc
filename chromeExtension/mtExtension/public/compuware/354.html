Importing a Dump
Note:
This keyword is valid for viewing servers only.
You can import dumps directly into an Abend-AID viewing server from the MVS console,
using the MVS MODIFY command and its IMPORT keyword. Further, if the dump being
imported contains one and only one CICS address space, the Abend-AID Viewer
automatically starts dump analysis once it has imported the dump if the Auto Import
field for the region dump profile member that matches the dump is set to Y. Refer to
“Modifying Region Dump Profiles” on page 20-14 for more information about the region
dump profile.
Note:
You can also use the Abend-AID Viewer SVC dump import function or the MVS
post-dump exit to import dumps.
The following syntax is valid for the MODIFY command:
jobname
The viewing server job name.
datasetname
The name of the dataset to import into the Abend-AID Viewer. You can specify the
dataset name with or without quotation marks.
Displaying Viewing Server, BDCAS, or TDCAS Information
The MVS MODIFY command allows you to display information about the viewing server
BDCAS, or TDCAS.
The following syntax is valid for the MODIFY command:
jobname
The viewing server, BDCAS, or TDCAS job name
Users
Displays all users logged onto the viewing server or the user ID assigned to the
regions attached to the BDCAS or TDCAS.
