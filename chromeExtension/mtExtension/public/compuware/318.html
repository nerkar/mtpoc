Notify Dups
Specifies whether to notify the user of duplicate dumps. Valid values are:
N:
Default. Do not notify regarding duplicate dumps.
Y:
Notify regarding duplicate dumps.
Logon
Specifies whether to add the LOGON parameter to the MVS SEND command used for
the notify request. Valid values are:
N:
Default. Do not add the LOGON parameter to the MVS SEND command.
Notify the user only if he or she is logged onto the system at the time the dump
is taken.
Y:
Save the notification in the SYS1.BRODCAST dataset and notify the user the
next time he or she logs onto the system.
The default value for the Logon field specifies that the notification is not saved in
the SYS1.BRODCAST if the user is not logged onto TSO when the notification is sent.
Example
Figure 21-6 shows three sample action definition entries for transaction dumps.
Figure
21-6.
Transaction Action Definitions Screen
The first entry specifies that TSO user ID CFXRJA0 is to receive notification for all abends
for all MVS SYSIDs, abend codes, and program names with the transaction ID COBI that
occur with the job name and APPLID H01AC122. When abends matching all these
criteria occur, the procedure named ABCOBOL is executed. This user is to be notified
when he or she logs onto remote SYSID CW02 and is to receive notification for duplicate
dumps.
The second entry specifies that TSO user ID CFXBKC0 is to receive notification for all
abends for all MVS SYSIDs, abend codes, transaction IDs, program names, and job names
that occur with APPLID H01AC060. This user is to receive notification when he or she
logs onto the MVS SYSID where the abend occurred and is not to receive notification for
duplicate dumps.
The third entry specifies that TSO user ID CFXSRE0 is to receive notification for abends
for all MVS SYSIDs, abend codes, transaction IDs, and program names that occur with the
job name and APPLID H01AC124. This user is to receive notification only if he or she is
logged onto the MVS SYSID where and when the abend occurred, and is not to receive
notification for duplicate dumps.
