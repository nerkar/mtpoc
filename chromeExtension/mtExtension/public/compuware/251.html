Configuring Regions
On the Summary Configuration screen (Abend-AID only installed) or the Region
Configuration screen (Abend-AID and Abend-AID for CICS or Abend-AID for CICS only
installed), you assign Abend-AID Viewer servers to process dumps for specific regions. A
combination of job name and MVS SYSID is used to identify each region. By default, all
dumps are processed by the very first viewing server ever started at your site.
You can also indicate which region dump profile member you want each region to use.
By default, all regions (qualified by job name and MVS SYSID) use the $DEFAULT
members for region dump profiles. If you want any regions to use values other than these
defaults, you must use the summary configuration function.
