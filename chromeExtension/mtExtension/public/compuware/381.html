Batch Abend-AID Utilities
E-3
Columns 11–18
Specify the table load module name. Do not specify a load module name when an
asterisk is entered in column 1. These are the standard tables:
Columns 19–25
For translation tables only. Place a comma in column 19, and then specify one of two
formats. The entry FORMAT obtains a formatted display of the table. DUMP obtains a
hexadecimal and character dump of the table. You can also obtain a dump by leaving
column 19 blank.
Examples
H CWCMTRHT,DUMP OPTIONAL HORIZONTAL TRANSLATION TABLE
H CWCMTRVT,FORMAT OPTIONAL VERTICAL TRANSLATION TABLE
Columns 60–65
Indicate whether the table is required (Y) or not (N). Default optional tables may be made
required. An optional specification for a default required table, however, will be
overridden. Report warnings with non-zero return codes notify you of a required table
not found or an overridden optional specification.
