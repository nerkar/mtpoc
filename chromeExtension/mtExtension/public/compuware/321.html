The first entry specifies that TSO user ID CFXAMJ1 is to receive notification for all dumps
for all APPLIDS, MVS SYSIDs, and abend codes that occur with the job name CFXAMJ1.

dumps for all APPLIDs, MVS SYSIDs, and abend codes that occur with the job name
CFXPMK0. This user is to receive notification only if he or she is logged onto the MVS
SYSID where and when the abend occurred.
Specifying Contact Information
Note:
This option is available only if Abend-AID for CICS is installed on this viewing
server.
Once you have created a transaction or region dump action definition entry (formerly
known as a
notification
entry), you can specify detailed contact information for that entry
on the Contact Information screen, as shown in Figure 21-9 on page 21-10. Users can
access the information that you input here by entering the
N
line command next to the
entry for any dump on the Abend-AID Directory that matches all of the criteria of the
action definition.
To access the Contact Information screen, enter the
C
line command next to a
transaction or region dump action definition entry on the Transaction Action Definitions
or Region Action Definitions screen.
