Specifying the Default Site Screen Attributes
This function is used to set the default Abend-AID Viewer screen attributes for all users at
your site. If you don’t modify any of the default screen attribute values, the Compuware-
supplied defaults are used. Individual users can override the values you set here when
they sign onto the Abend-AID Viewer. The screen attributes function is documented in
the
Abend-AID User/Reference Guide
or the
Abend-AID for CICS User’s Guide
.
Note:
Changes you make on this screen do not affect the customization screens.
