Adding a Profile Definition for the AADF Transaction
The AADF transaction requires a PROFILE option so that the alternate screen size can be
used when viewing reports from CICS. The default profile name is
CCAAPRO2
. You can
change the profile name to conform to your site standards, as long as you also change the
name in the AADF transaction. If you rename the AADF transaction and the CCAAPRO2
profile, make sure you change the profile name in the renamed transaction. Do not
change the associated program.
The CCAAPRO2 profile contains the same values for the operands as the IBM-supplied
DFHCICST profile with the exception of the SCRNSIZE value, which is set to ALTERNATE.
The profile definition for CCAAPRO2 is contained in each RDO member in the
installation sample library (CTL).
