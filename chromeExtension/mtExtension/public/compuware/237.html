Workbench Host/Port Table Screen
This table is used when viewing an Abend-AID report in the Topaz Workbench and a
request needs to be made to the Workbench, for example, for File-AID support. The SMF
SYSID for the report being viewed is used to look up in this table the appropriate host
name and port number to establish a connection to the Workbench.
Authorized users can use the following line commands that are valid for this screen:
•
A (Add entry):
Use this line command to display the Add Host/Port Information
screen (shown in Figure 18-8), which lets you add a table entry. After you input the
information, use the END (PF3) key to save your inputs and return to the Workbench
Host/Port Table screen (Figure 18-7), or enter the CANCEL primary command to exit
the screen without saving your inputs.
•
S (Update entry):
Use this line command to display the selected entry on the Update
Host/Port Information screen (shown in Figure 18-9 on page 18-6), which lets you
update any of the prefilled input fields. After you make your changes, use the END
(PF3) key to save your changes and return to the Workbench Host/Port Table screen
(Figure 18-7), or enter the CANCEL primary command to exit the screen without
saving your changes.
•
D (Delete entry):
Use this line command to delete the selected table entry.
