Add a DD statement for the demonstration file to the CICS region JCL, if you did not

You can remove the demonstration file DD statement when the CCAADEMO,
CFDB2IVP, CFDB2IV2, CCAADEMP, and CCAADEMA programs are no longer
required for demonstration purposes.

CFDB2IVP is returned a -818 or -805 SQLCODE, resulting in an IVPA

variable and its data type.
