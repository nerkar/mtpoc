Site Configuration Examples
D-9
After you create your first viewing server, you can easily configure additional viewing
servers by repeating the steps described in Chapter 3, “Abend-AID Viewer
Configuration”.
Figure D-4 illustrates the dataset usage of an Abend-AID configuration for SERVER1
running on MVS1, and SERVER2 running on MVS2.
Figure
D-4.
Dataset Usage of Two Viewing Servers on Two MVS Images
Notes:
1.
The customization file and system DSECT file are allocated on shared DASD, and are
used by both SERVER1 and SERVER2.
Ensure that these files are part of a GRS ring to protect their integrity. See “GRS
Considerations” on page 15-26 for more information.
The user DSECT file and source listing files are not shown. If you are using these
facilities, these files can be shared by both viewing servers (if they are allocated on
shared DASD), or each viewing server can have its own copy of the files.
2.
Each viewing server has its own set of the following
viewing server-owned
datasets:
–
Dump information file
–
IPCS directory (optional)
–
PDSM file
–
Viewing Server work file
–
Shared directory
–
Transaction databases. Compuware recommends that you allocate at least two
transaction databases per viewing server.
