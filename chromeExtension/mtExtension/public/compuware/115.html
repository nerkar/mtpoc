Controlling Report Format
Viewing or printing Abend-AID reports can be done in a narrow (72-column) or wide
(121-column) format when routed to SYSOUT or a sequential file. The report format in
the Abend-AID report dataset cannot be altered. Refer to Table 9-1 to determine the
column width of the Abend-AID report.
