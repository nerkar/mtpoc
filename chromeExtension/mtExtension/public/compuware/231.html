Step 4. Modify and Copy the MSDCCOPY Procedure
Modify the procedure contained in installation sample library (INS) member MSDCCOPY
and copy it to a system PROCLIB.
Ensure that:
•
The STEPLIB DD statement points to your Abend-AID Viewer authorized load library
(SKAZAUTH).
•
The FDBDCUST DD statement points to the Abend-AID Viewer customization file for
this release.
Note:
If you are using an external security package, perform the security functions
described in “Authorizing the MSDCCOPY Procedure” on page 23-4.
Step 5. Enable the Post-Dump Exit
To enable the post-dump exit before you perform an IPL, refresh the link list via an LLA
REFRESH console command and cycle DUMPSRV via a CANCEL DUMPSRV console
command. DUMPSRV is immediately automatically restarted by MVS.
Step 6. Modify the Region Dump Profile screen
After you complete the previous steps, use the Region Dump Profile screen to indicate
which SVC dumps are to be automatically copied and imported into the Abend-AID
Viewer. The Region Dump Profile screen is described in “Modifying Dump Capture
Profiles” on page 20-13.
If you want to use the notification facility for region SDUMPs, create appropriate
notification definitions, as described in Chapter 21, “Action Definitions”.
