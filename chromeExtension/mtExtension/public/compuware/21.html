Receive From Network (RFN) Installation
Chap 1
The Receive From Network (RFN) installation process for Compuware mainframe
products has been enhanced to use the Compuware Installer to generate your site’s
installation JCL. The RFN process enables you to receive the products directly to your
z/OS environment and install them using System Modification Program Extended
(SMP/E).
After you select a licensed product from Compuware’s online ordering system, you will
receive an e-mail message containing sample JCL for creating a Compuware Installer
dialog PDS, which has been customized for your order.
Documentation for using the Compuware Installer to install one or more licensed Abend-
AID components (Abend-AID, Abend-AID for CICS, and Abend-AID Common
Components) is provided in the
Compuware Installer Mainframe Products SMP/E Installation
Guide
PDF
document included with your order e-mail. The e-mail may also contain links
for downloading any PC-based products that may have been ordered at the same time.
Alternatively, if for some reason you can’t do an RFN installation at your site and you
ordered the Abend-AID EP media interface, you can obtain a copy of this document from
Compuware GO (
http://go.compuware.com
) in the ECC documentation section. The
Compuware Installer document also contains information for using the EP physical
media to SMP/E install the Abend-AID components.
The complete RFN installation using SMP/E for Abend-AID Release 16.05 consists of the
following three product components:
•
Batch Abend-AID, FMID: MPAA160
•
Abend-AID for CICS, FMID: MKFX161
Support for CICS Transaction Server 5.3, FMID: NKFX168
Support for CICS Transaction Server 5.2, FMID: NKFX167
Support for CICS Transaction Server 5.1, FMID: NKFX166
Support for CICS Transaction Server 4.2, FMID: NKFX165
Support for CICS Transaction Server 4.1, FMID: NKFX164
•
Abend-AID Common Components, FMID: MKAZ160
If your site is licensed, you may install batch Abend-AID or Abend-AID for CICS or both,
and Abend-AID Common Components is required for use with either.
Notes:
1.
Abend-AID Release 16.05 requires Enterprise Common Components (ECC) Release
16.05.
2.
Enterprise Common Components Release 16.05 cannot be in the same zones as
Abend-AID 12.4 and prior releases.
3.
If you plan to install the Abend-AID Fault Analytics option, it has been packaged
with Abend-AID Common Components and therefore requires no separate SMP/E
installation.
