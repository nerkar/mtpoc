Step 2. SMP/E RECEIVE the FMID and Preventive Service
Installation sample library (CTL) member CxxxRECV contains JCL to perform an SMP/E
RECEIVE of the Abend-AID for CICS FMID and preventive service for the level of CICS
support you are installing. These tasks are performed with separate job steps. Ensure that
the //SMPPTFIN DD statement in the second job step points to the correct maintenance
file.
Notes:
1.
If your RFN installation package for this Abend-AID for CICS release included the
FMID for the CICS release you want to add, you may skip this step and proceed to
Step 3.
2.
Run CxxxRECV in a job class suitable for long-running jobs.
3.
If there is no preventive service available for the specified FMID(s), the second step in
CxxxRECV returns message
GIM24801W No SYSMODS satisfied the operands
specified on the receive command
.
4.
All existing FMIDs must be at the current maintenance level prior to adding support
for a new CICS release.
Step 3. SMP/E APPLY the FMID and Preventive Service
Installation sample library (CTL) member CxxxAPLY contains sample JCL to SMP/E
APPLY the Abend-AID for CICS FMID and preventive service for the level of CICS support
you are installing. Before you submit CxxxAPLY, verify that the APPLY SELECT statement
is applying the correct FMID.
Notes:
1.
Run an APPLY CHECK before applying the FMID.
2.
Run CxxxAPLY in a job class suitable for long-running jobs.
Step 4. SMP/E ACCEPT the Abend-AID for CICS FMID
Installation sample library (CTL) member CxxxACPT contains sample JCL to SMP/E
ACCEPT the Abend-AID for CICS FMID for the level of CICS support you are installing.
Before you submit CxxxACPT, verify that the ACCEPT SELECT statement is accepting the
correct FMID.
Notes:
1.
Preventive service is not accepted by default. Thoroughly test all PTFs before you
accept them.
2.
Run an ACCEPT CHECK before accepting the FMID.
3.
Run CxxxACPT in a job class suitable for long-running jobs.
