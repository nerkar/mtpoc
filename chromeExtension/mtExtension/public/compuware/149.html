Stopping HOTKEY Processing
If you need to stop HOTKEY processing only:
1.
Execute only the
first
step of sample JCL in installation sample library (CTL) member
JCLASCNL.
2.
Ensure that the OKPARM is set to 'REQ=STOP'.
If you need to stop both the HOTKEY and URLPUB functions and bring down the
HOTKEY/URLPUB address space:
1.
Execute
both
steps of sample JCL in JCLASCNL.
2.
Ensure that the OKPARM is set to 'REQ=STOP'.
3.
Ensure that the UPPARM is set to 'REQ=STOP'.
Note:
This job stops the HOTKEY/URLPUB address space only. See Appendix C,
“Controlling the Abend-AID Subsystem” for more information about the Abend-
AID subsystem.
Enabling Batch Abend-AID to Publish URL to Fault Analytics
Notes:
1.
Refer to the Abend-AID Fault Analytics installation guide for more information about
installing this product.
2.
Optional URLPUB support initialization is required for batch Abend-AID sites only.
Abend-AID for CICS sites have this support without performing this step.
3.
Optional URLPUB and HOTKEY support should use the same Abend-AID subsystem
that is used by the BDCAS and/or TDCAS jobs on this MVS image. URLPUB and
HOTKEY support share the address space created during HOTKEY and/or URLPUB
initialization. See “Optional HOTKEY Pop-Up Window Support Initialization” on
page 11-2.
Batch Abend-AID can publish the report URL to Abend-AID Fault Analytics Release 4.1 or
more current. URLPUB support links batch Abend-AID and Abend-AID Fault Analytics via
Web Browser Interface viewing access. When batch Abend-AID sends an event to Fault
Analytics, URLPUB support includes the link from Fault Analytics in the Abend-AID
report in the Web Browser Interface viewing access. Your site’s Fault Analytics users can
click on the link and open the report in the browser-based viewing access. See
“Configuring Abend-AID Web Browser Interface Access” on page 16-20 for more
information.
URLPUB support requires the installation of Abend-AID Common Components and is
performed by installation sample library (CTL) member JCLURLPB. If your site plans to
use this optional support, run JCLURLPB. This job contains steps to start an Abend-AID
subsystem and perform the URLPUB initialization.
