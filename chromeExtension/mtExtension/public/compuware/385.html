CWMODLST List Abend-AID Modules JCL (JCLMODLS)
Verify that the dataset name entered in the CWNAMES DD statement contains the
module names.
CWMODLST syntax
[PARM = ‘‘ (none)]
PARM = ‘PLI’
PARM=’DATE=dd mmm yyyy’
none
Lists all Abend-AID modules installed. Generates the standard Abend-AID report.
DATE
Lists all modules installed after the specified date, or found in error.
PLI
Lists Abend-AID PL/I support modules only.
JCLSMPDD
JCLSMPDD (formerly, $08SMPDD) is a sample JCL member that performs an SMP/E LIST
DDDEF to list the DDNAMEs and DSNAMEs in the SMP/E target zone. The SHASMAC
DDNAME found in the output of this job corresponds to the DSNAME of the correct JES2
macros.
Use this DSN name to successfully generate $09ASMJT. This JCL creates the JES2 Control
Block Offset table, as described in “1. Execute $09ASMJT for JES2 from the CTL Library”
on page 4-4.
Verify your job card, z/OS SMP/E global CSI dataset, and add your site's z/OS target zone
name prior to running this JCL.
