Suppress Abend-AID for Started Tasks

Maximum element display for LE storage
This option limits the number of LE heap elements analyzed by Abend-AID.
Maximum SAS/C CSA length displayed
This option sets the length in bytes to use to format a SAS/C C program save area in
the event that Abend-AID cannot determine the length by usual means.
Maximum DSAs displayed for LE U4083
This option limits the number of LE Dynamic Save Areas (DSAs) formatted by Abend-
AID during the analysis and diagnosis of an LE U4083 abend.
