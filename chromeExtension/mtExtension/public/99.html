Abend-AID, Abend-AID for CICS, and Abend-
AID Viewer Customization
Abend-AID, Abend-AID for CICS, and Abend-AID Viewer customization is optional,
depending on your site’s requirements for use of the products. Part 3 contains the
following chapters:
Chapter 8, “Batch Dump Capture Address Space Customization”:
Chapter 8 describes the steps for configuring the batch dump capture address
space (BDCAS), which is required for use with batch Abend-AID.
Chapter 9, “Batch Abend-AID Processing Control”:
Chapter 9 describes the batch Abend-AID processing control hierarchy and
how to customize the optional DD statements and tables. It also discusses
how to use the batch Abend-AID customization dialog to customize the
required tables, CWROUTE and CWGLOBAL.
Chapter 10, “Batch Abend-AID User-Coded Customization”:
Chapter 10 describes user-coded diagnostics, help text, and the user exits.
Chapter 11, “Batch Abend-AID Additional Facilities and Considerations”:
Chapter 11 describes the additional facilities and considerations for batch
Abend-AID customization.
Chapter 12, “Transaction Dump Capture Address Space Customization”:
Chapter 12 describes the steps for customizing the transaction dump capture
address space (TDCAS), which is required for use with Abend-AID for CICS.
Chapter 13, “Abend-AID for CICS Additional Facilities and Considerations”:
Chapter 13 descibes Abend-AID for CICS additional facilities and
considerations.
Chapter 14, “Abend-AID for CICS User-Defined Diagnostics”:
Chapter 14 describes how to code text modules for Abend-AID for CICS site-
specific transaction abends.
Chapter 15, “Viewing Server Customization”:
Chapter 15 describes how to modify Abend-AID viewing servers.
Chapter 16, “Viewing Access Customization”:
Chapter 16 describes how to modify TSO/ISPF, CICS, VTAM, or Web Browser
Interface viewing access. It also includes the steps for installing Distributed
Viewing Support (DVS).
