Site News Menu
Select either option 1 or 2 from the Site News Menu, to display the Add/Maintain Site
News input screen for Abend-AID or Abend-AID for CICS, respectively, as shown in Figure
18-4. The text is displayed as entered without any formatting. The site-specific news
resides on the customization file so all viewing servers using the same customization file
have access to the same text.
Figure
18-4.
Add/Maintain Site News Screen
To delete previously created site news, select option 3 or 4 from the Site News Menu to
display the Confirm Delete for News window for Abend-AID or Abend-AID for CICS,
respectively. Press Enter to complete the deletion. Press the END key (PF3) to cancel the
delete request.
Creating/Maintaining User-Defined Diagnostics
The User-Defined Diagnostics screen, shown in Figure 18-5 on page 18-4, enables
authorized users to create or maintain user-defined diagnostic entries. Users are granted
security authorization to this screen with access to the Customizations Options menu
(Figure 18-2 on page 18-2) through the CUSTOM subparameter of the
EXTERNAL_SECURITY_FUNCTION_CHECK viewing server configuration parameter, as
described in “Parameters to Enable External Security for Viewing Servers” on page 15-8.
The user-defined diagnostics text resides on the customization file so all viewing servers
using the same customization file have access to the same text.
